Current version: 0.1

Clan Comet is a content management system aimed at esport teams who need powerfule website solution. It's made on Codeigniter PHP framework, and comes with a ton of features!

Installing:
1. Upload all the files on server.
2. Visit: www.yoursite.com/install
3. Follow the steps

Codeigniter plugins:
* twiggy parser
* hmvc
* ion auth
* template
* html purifier
* php markdown

HTML / CSS / JS:
* Morris.js
* CKEditor
* MarkItUp
* Pick Date
* Twitter Bootstrap
* Font Awesome Icons
* Select 2
* Bootbox
* iCheck
* jQuery Cookie
* SlimBox 2
